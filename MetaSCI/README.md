### Directory

- `dataset\mask` : encoding mask, shape = [M N Cr NUM], i.e., NUM  [M N Cr] masks stacked along the last dimension.





### Parameter

- `datadir`:  path for training set directory
- `maskpath`: path for encoding mask  file
